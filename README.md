## 🚀 Introduction

SkillScan is a high-performance, AI-driven platform designed to help job seekers optimize their resumes for Applicant Tracking Systems (ATS). Using advanced AI models, SkillScan analyzes your resume against specific job descriptions to provide detailed feedback, ATS scores, and actionable improvement tips.

## 🔋 Key Features

- **⚡ Fast AI Analysis**: Optimized with GPT-4o-mini for near-instant feedback.
- **📊 Job Match Likelihood**: Visualized probability graph showing your chances of landing the interview.
- **📄 Instant PDF Preview**: High-quality resume preview generation for easy review.
- **🔒 Serverless Auth & Storage**: Privacy-first authentication and file storage powered by Puter.js.
- **📱 Responsive Design**: Modern, sleek UI that works perfectly on any device.
- **🎯 Precise ATS Scoring**: In-depth evaluation based on tone, content, structure, and skills.

## ⚙️ Tech Stack

- **React & React Router v7**: For a seamless, dynamic single-page application experience.
- **Puter.js SDK**: Handles serverless auth, file storage, and AI interactions entirely on the client side.
- **Tailwind CSS**: Modern utility-first styling for a premium look and feel.
- **TypeScript**: Ensuring high code quality and type safety.
- **Vite**: Ultra-fast build tool and development server.

## 🤸 Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone the repository:
   ```bash
    git clone git@github.com:pariharakanshi33-svg/SkillScan.git
    cd SkillScan
    ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
